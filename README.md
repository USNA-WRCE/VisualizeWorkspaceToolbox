# VisualizeWorkspaceToolbox
MATLAB Toolbox for Visualizing the Reachable and Dexterous Workspace of Articulated Robotic Manipulators

## First Time Installation Instructions
1. Download "VisualizeWorkspaceToolbox.zip" (or alternate version)
2. Unzip/Extract "VisualizeWorkspaceToolbox.zip"
3. Open MATLAB as an administrator
4. Change your MATLAB Current Directory to the location containing contents of the unzipped TurretSimToolbox
5. Run "installVisualizeWorkspaceToolbox"

## Update Instructions
1. Open MATLAB as an administrator
2. Run "VisualizeWorkspaceToolboxUpdate"
